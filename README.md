# RapRapesYourEars
Study the evolution of the languages used into french rap lyrics.


## Installation
`> mkvirtualenv RapRapesYourEars`
`> pip install bs4`
`> pip install pymongo`


## Usage

### Launch mongo db
`> mongod`


### Execute script
`> cd RapRapesYourEars`
`> workon RapRapesYourEars`
`> python script.py`
